# 🐦‍🔥 Феникс (Phoenix Space) — Твой Цифровой Ковчег

**Инструмент для суверенной цифровой жизни. Открытый, некоммерческий, безопасный.**

«Феникс» даёт тебе три вещи, которые отняли корпорации и государства: **контроль над своей личностью, приватное общение и вечное хранилище для творчества.**

---

## 🛡️ Почему это важно
- **Твой DID (Цифровой Паспорт):** 12 слов, которые генерируются и хранятся только у тебя. Никакой сервер не может заблокировать твой аккаунт.
- **Прямая связь (P2P):** Звонки и сообщения идут напрямую собеседнику, без посредников. Никаких логов, никакой прослушки.
- **Творчество навсегда (NFT):** Твои артефакты записываются в блокчейн и остаются с тобой навсегда.

Мы не собираем данные, не показываем рекламу, не манипулируем вниманием.

---

## 🗺️ Текущее состояние
- **Готово:** DID, Академия, Мастерская, P2P-чат, мультиязычность, адаптивный дизайн.
- **В процессе:** P2P-звонки, передача файлов, расширение контента.
- **План:** Сеть Садов (полная децентрализация), Phnx-токены (экономика вклада).
