# 🐦‍🔥 Феникс: Дорожная карта

Мы строим единое приложение на сервере 103.76.54.130. Это наш живой Сад.

### ✅ Пройденные этапы
- Основа: Пульсар, лучи, Академия, Живое Поле.
- Контент: Шесть Врат, практики, сутры, админ-панель.
- Доступ: Чистый URL (80 порт), PWA для установки на устройства.

### 🚀 Ближайшая цель: Пространство для практик и общения
1.  Добавить в Академию зал «Круг Соратников» (текстовый чат/форум).
2.  Создать зал «Мастерская» (возможность загружать свои артефакты — текст, аудио).

- Единая сеть Мастеров, где каждый ведёт свой Сад.
