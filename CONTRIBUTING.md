# 🤝 Как внести вклад в «Феникс»

Спасибо, что решили помочь! Мы строим некоммерческий, открытый инструмент для цифровой свободы. Вот как можно присоединиться.

## 🛠️ Быстрый старт (локально)
1.  **Склонируйте репозиторий:** `git clone https://github.com/v4g7yz29g7-coder/phoenix-space.git`
2.  **Установите зависимости:** `cd phoenix-space && npm install`
3.  **Запустите сервер:** `node server.js`
4.  Откройте `http://localhost:3000`

## 📋 Как помочь
- **Разработчики:** Посмотрите открытые задачи (issues) или предложите свои улучшения через Pull Request.
- **Авторы:** Дополните Академию новыми сутрами или практиками (файлы в `public/`).
- **Дизайнеры:** Помогите улучшить UI/UX, адаптивность под мобильные устройства.
- **Тестировщики:** Сообщайте о багах через раздел Issues.

## 📝 Правила
- Придерживайтесь стиля кода, уже используемого в проекте.
- Для крупных изменений сначала создайте обсуждение (issue).
- Будьте вежливы и уважайте чужой труд.
